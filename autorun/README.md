# aliases-for-windows

## How to use
1. Double click the `register-aliases.reg` file.
2. Click 'Yes' on the User Account Control window that shows up.
3. You're all done

## How to un-use?
1. Double click the `unregister-aliases.reg` file.
2. Click 'Yes' on the User Account Control window that shows up.
3. You're all done
